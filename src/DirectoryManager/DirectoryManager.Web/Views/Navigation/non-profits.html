Non-Profits › Operating Systems

Learn about non-profit operating systems that enhance Monero’s privacy and security.

Advertise here

  • Artix Linux

    Rolling-release distribution based on Arch Linux that uses inits such as OpenRC, runit, s6 or Dinit.

  • CalyxOS

    An Android mobile OS that puts privacy and security into the hands of everyday users. (Note: see donations page and Calyx Institute)

  • GrapheneOS

    Private and secure mobile operating system with Android app compatibility.

  • Kicksecure

    A secure-by-default OS always based on the latest security research.

  • Linux Mint

    An OS for desktop & laptop, designed to work out-of-the-box with apps most people need.

  • Qubes

    A reasonably secure operating system.

  • Tails

    Privacy-focused operating system.

  • Whonix

    Software that can anonymize everything you do online.
